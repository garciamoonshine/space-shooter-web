# 🚀 Space Shooter Web

A classic arcade-style space shooter built with HTML5 Canvas and vanilla JavaScript.

## Features
- Player ship with smooth movement
- Multiple enemy types (A, B, C) with different behaviors
- Boss battles every 5 levels
- Power-ups: Shield, Rapid Fire, Multi-Shot, Bomb
- Particle explosion effects
- Progressive difficulty scaling
- Lives system and high score tracking
- Mobile touch support

## Controls
| Input | Action |
|-------|--------|
| ← → | Move ship |
| Space | Shoot |
| Z | Bomb |
| Touch | Tap sides to move, tap center to shoot |

## Tech Stack
- HTML5 Canvas API
- Vanilla JavaScript (ES6 classes)
- No dependencies
